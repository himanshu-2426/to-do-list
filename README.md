# Todo List Project

A simple browser-based todo list app built with HTML, CSS, and JavaScript.

## Features
- Add new tasks
- Mark tasks as completed or undo them
- Delete tasks
- Save tasks locally in the browser

## Project Structure
- `index.html` - Main app structure
- `style.css` - Styling for the interface
- `script.js` - App logic for adding, completing, deleting, and saving todos

## How to Run
1. Open `index.html` in your browser.
2. Start adding tasks to your todo list.

## Notes
Tasks are stored in the browser's local storage, so they remain available after refreshing the page.
